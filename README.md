##判断闰年
  <br>编写Java程序，输出从公元1990年到2007年所有\<br>
  <br>闰年的年号，每输出两个年号换一行。判断年号是</br>
  <br>否为闰年的条件是：</br>
  <br>（1）若年号能被4整除，而不能被100整除，则是闰年；</br>
      （2）若年号能被400整除也是闰年。
